# 连点器 — 需求文档

> 版本：v1.0  
> 最后更新：2026-05-24  
> 状态：已确认（用户签字）

---

## 1. 项目概述

开发一款运行在 Windows 平台上的轻量级连点器桌面应用。用户可通过简洁的图形界面配置要连点的键（鼠标或键盘）和时间间隔，启动后程序按设定间隔自动重复点按，直到手动停止。程序在后台最小化运行仍可正常工作。

---

## 2. 功能需求

### 2.1 核心功能

| 编号 | 功能 | 说明 |
|------|------|------|
| F-01 | 选择连点键 | 支持鼠标键（左键、右键、中键、侧键 X1/X2）和键盘任意键 |
| F-02 | 设置时间间隔 | 以毫秒为单位输入 |
| F-03 | 开始连点 | 点击「开始」按钮后按设定间隔重复点击 |
| F-04 | 停止连点 | 点击「停止」按钮后立即停止 |
| F-05 | 后台生效 | 窗口最小化或失去焦点时，连点功能持续运行 |

### 2.2 键选择方式

- **键盘键**：通过「捕获按键」模式选择。点击「捕获按键」按钮后，按下键盘任意键即记录该键
- **鼠标键**：通过下拉框选择（左键 / 右键 / 中键 / 侧键 X1 / 侧键 X2）

### 2.3 运行行为

- 不限制连点次数，持续运行直到手动停止
- 连点过程中可以修改键和间隔设置（修改后立即生效？还是下次启动生效？——待定）
- 停止后可以再次开始，无需重启软件

---

## 3. 非功能需求

| 编号 | 需求 | 说明 |
|------|------|------|
| NF-01 | 轻量 | 单 exe 文件，双击即用，无需安装运行时环境 |
| NF-02 | 低资源占用 | 运行时不占用过多 CPU 和内存 |
| NF-03 | 响应式 | 开始/停止响应迅速，无卡顿 |
| NF-04 | 稳定性 | 长时间运行不崩溃 |

---

## 4. UI/UX 需求

| 编号 | 需求 | 说明 |
|------|------|------|
| UX-01 | 简洁直观 | 界面元素精简，功能一目了然 |
| UX-02 | 主色调淡蓝色 | #D4E8F7 或类似色系 |
| UX-03 | 窗口尺寸适中 | 固定尺寸小窗口，不拉伸 |

---

## 5. 交付要求

- 打包为单个 `.exe` 文件
- 支持 Windows 10 / 11

---

## 6. 术语表

| 术语 | 说明 |
|------|------|
| 连点 | 以固定时间间隔自动重复执行按键操作 |
| 间隔 | 两次按键之间的等待时间，单位为毫秒（ms） |
| 捕获按键 | 点击按钮后按下任意键，程序自动记录该键值 |
