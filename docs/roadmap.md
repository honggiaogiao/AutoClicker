## 开发原则

1. **小步迭代**：每完成一个步骤都确保可验证、可运行
2. **先核心后界面**：先让连点逻辑跑通，再美化界面
3. **随时可打包**：每个里程碑完成后都可以打包成 exe
4. **错了能回退**：每步改动前确保 git（或手动备份）

## 阶段三：核心连点逻辑

| 步骤 | 内容 | 说明 |
|------|------|------|
| 3.1 | 实现键盘连点模块 | 一个独立模块，使用 keyboard 库模拟按键 |
| 3.2 | 实现鼠标连点模块 | 使用 mouse 库模拟点击 |
| 3.3 | 实现线程管理 | 连点运行在独立线程中，支持开始/停止控制 |
| 3.4 | 单元验证 | 写一个简单的控制台测试，验证连点功能正常 |

### 注意要点
- 连点线程需能安全停止（使用 Event 信号而非强制终止）
- 间隔控制使用 `time.sleep` 或精确计时
- 鼠标点击区分左右键
